Introduccion

El proyecto consta de un brazo diseNado a partir de 6 servomotores controlados por una Web a partir de TCP/IP por medio del ESP32, a lo largo del mismo nos encontramos con varios desafios que fuimos resolviendo mediante pruebas e investigacion, incorporando asi conocimientos de seNales de pulso, niveles logicos, arquitectura de un servomotor, soldadura, circuitos electronicos, entre otros tantos. El mismo nos hizo integrar varias areas de la industria 4.0, tales como protocolos de comunicacion, control de motores, diseNo Web, impresion 3D y PWM. El modelo del brazo es EEZYbotARM MK2. Este modelo lo podemos descargar desde Thingiverse e imprimirlo con una impresora 3d. El servidor web es basado en un proyecto de FABRI creato, el cual fue modificado para agregar los servos correspondientes a cada eje.
